CHAOS

Commands List

Insert your prefix before each of the commands (e.g. if your prefix is > do >ping etc.

[ ] = optional, < > = required.


AFK
Aliases: afk, away
Usage: >afk [Message]
Description: Auto replies to mentions when you are afk.

Avatar
Aliases: avatar, pfp, av
Usage: >av [@User/ID]
Description: Sends your avatar or a mentioned users avatar.

Backup
Aliases: backup, bak
Usage: >backup
Description: Backs up your discord account in json format.

cat
Aliases: cat, meow
Usage: >cat
Description: Sends a random cat image!

Clear DMs
Usage: >cdms
Description: Closes dm channels that you haven't spoken in in a month

Clear
Usage: >clear
Description: Clears the CHAOS console.

Help
Aliases: help, cmds, commands
Usage: >help [command]
Description: Sends the command list URL OR Displays info about a specified command.

Coin
Usage: >coin
Description: Flips a coin (Sends heads or tails).

Dice
Aliases: dice, d6
Usage: >dice
Description: Rolls a dice (Sends a random number from 1 - 6).

Theme
Aliases: theme, dth
Usage: >theme 
Description: Changes your discord theme.

SysFetch
Aliases: sysfetch, sysinfo, fetch
Usage: >sysfetch
Description: Sends your system information (specs).

Figlet
Aliases: figlet, fig
Usage: >fig 
Description: Turns your text into an ascii figlet.

HypeSquad
Aliases: hypesquad, house, hs
Usage: >hypesquad 
Description: Changes your HypeSquad house.

IdToDate
Aliases: idtodate, idtod, idtd
Usage: >idtodate 
Description: Converts any discord id to a normal date format.

IP
Usage: >ip 
Description: Looks up information about a specified IP/URL.

Ping
Aliases: ping, latency
Usage: >ping [IP/URL]
Description: Checks how long it takes to connect to the discord gateway OR a specified IP or URL in milliseconds.

LeaveGroupDms
Aliases: leavegroupdms, lgdm, leavegroups
Usage: >lgdm
Description: Leaves all group DMs.

MuteAllServers
Aliases: muteallservers, mas, muteservers, muteguilds, muteallguilds, mag
Usage: >mas
Description: Mutes all servers indefinitely.

Pride
Usage: >pride <@User>
Description: Overlays the lgbtq flag over someones pfp.

Purge
Aliases: purge, sclear
Usage: >purge [Channel] 
Description: Deletes a specified amount of YOUR messages from a channel.

Quit
Aliases: quit, q, selfdestruct
Usage: >quit
Description: Exits CHAOS.

Random Choice
Aliases: randomchoice, rchoice, rch
Usage: >rch 
Description: Picks a random option/choice out of a list you provide.

Restore
Usage: >restore
Description: Restores your account from a backup json.

Spam
Usage: >spam  
Description: Spams messages in a channel.

SPurge
Usage: >spurge
Description: Stops the purger.

SSpam
Usage: >sspam
Description: Stops the spammer.

Status
Aliases: status, st
Usage: >status 
Description: Changes your current status.

Token
Usage: >token
Description: Logs your token into the console.

update
Aliases: update, updoot, upgrade
Usage: >update
Description: Checks for CHAOS updates.

Upload
Usage: >upload 
Description: Uploads a file to 0x0.st.

Uptime
Usage: >uptime
Description: Displays how long CHAOS has been running for in Days, Hours, Minutes and Seconds.

UserInfo
Aliases: userinfo, user
Usage: >user [@User/ID]
Description: Displays information about a mentioned user, or yourself.
